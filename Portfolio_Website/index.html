ABOUT ME

Coding is something that has always interested in me. I enjoy the process of typing instructions in, running the program, seeing the results, going back and modifying something, and watching the changes take place. I am creative and enjoy the ability to create things from scratch using code. I first learned to program using BASIC on an Apple IIe when I was in middle-school, had some exposure to C++ when I was getting my CS degree, have written some database applications with MS Access utilizing Visual Basic code, have done shell scripting in Unix, and have created a number of personal websites using HTML, CSS, and just a bit of JavaScript. Every time I take on a new coding project, I realize just how much I enjoy it, which is why I am studying software development full-time at The Tech Academy.
